# プロンプト

## cloudに草案を作成してもらう

- プロンプト内容
```txt
Next.jsのプロジェクトからpayloadを入れる方法を簡単なブログ作成のCMSを作成したいです。
導入手順と作成手順を作成してください。
payloadの導入は「npm create payload-app@latest」からお願いします
技術は下記を使用してください。
・Next.js（最新）
・TypeScript 
・TailwindCSS 
・shadcn/ui 
・payload(template:blank)
・PostgreSQL

機能は下記の実装をしたいです。
・管理画面で記事の作成
・フロント画面にて表示されている記事に「イイね」ボタンの実装
・フロント画面にて表示されている記事にコメントする機能の実装
・「イイね」ボタンやコメントする機能は管理画面にて登録したユーザのみ大丈夫です

以上の要件でmarkdown形式で手順書を作成してください。
```
